# Example of a time series from carparts

A monthly time series from the `carparts` dataset, 51 observations, Jan
1998 - Mar 2002.

## Usage

``` r
carparts_example
```

## Format

Univariate time series of class [ts](https://rdrr.io/r/stats/ts.html).
